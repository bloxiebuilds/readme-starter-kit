## What is this?

**readme-starter-kit** is a collection of ready-to-use README building blocks for developers. No more staring at a blank file — just grab what you need and drop it in.

---

## 📦 What's Inside

| Block | Description |
|-------|-------------|
| 🏷 Badges & Shields | Status, version, license, and social badges |
| 📸 Screenshots | Placeholder layouts for previews |
| ✨ Feature Tables | Table and bullet versions |
| 🛠 Install & Setup | HTML, Node.js, and Python templates |
| 🤝 Contributing Guide | Fork, branch, PR instructions |
| 📋 Changelog | Version history template |
| 📄 License Footers | MIT, no-resell, and all rights reserved |
| ✏️ Full Template | A complete README starter you can build from |

---

## 🚀 How to Use

1. Open **[BLOCKS.md](./BLOCKS.md)**
2. Find the block you need
3. Copy the code
4. Paste it into your own README
5. Swap in your project details

That's it — no install, no setup, no dependencies.
